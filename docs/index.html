내 마음대로 배경화면

빠르고 쉽게 여러 장의 사진 또는 이미지를 배경화면으로 지정할 수 있는 어플리케이션!

여러 장의 배경화면 이미지 선택!

배경화면으로 사용할 여러 이미지를 선택하고, 미리보기, 확대보기 기능을 지원합니다.

배경화면 변경 시간 선택!

여러 장의 이미지를 배경화면으로 설정할 경우, 5분에서 24시간까지 변경 시간을 설정할 수 있으며, 스마트폰의 화면이 켜질 때마다 변경할 수 있는 기능을 지원합니다.

배경화면 검색 및 다운로드!

네이버 이미지 검색 API를 이용하여 쉽고 빠르게 배경화면을 다운로드합니다.

현재 설정된 배경화면의 정보를 한눈에 볼 수 있는 기능!

현재 배경화면으로 지정된 모든 사진을 순서대로 볼 수 있고, 스위치를 통해서 배경화면 설정을 해제하는 기능, 배경화면 변경 주기를 바꾸는 기능, 무작위로 배경화면을 변경하는 기능을 지원합니다.

소개 동영상

후기

처음에 생각했던 주제로 개발을 시작하지 못하고 중간에 주제를 변경했는데, 만족스러운 완성도를 보인 어플리케이션을 만들어서 뿌듯하다. 앱을 만들면서 가장 시간을 많이 쏟았던 부분은, 디바이스의 모든 사진들을 가져와서 화면에 출력 해주는 부분이다. 디바이스에 존재하는 모든 사진들을 가져와서 앱내에서 관리하는 것이 생각보다 메모리 누수가 많아서, Realm을 이용해서 디바이스 모든 이미지 파일들의 정보를 저장하였다. 그리고 두 달간 부스트캠프를 하면서 많은걸 배워갈 수 있어서 좋았다. MVP 패턴, Singleton 패턴(SharedPreferences와 MediaScannerConnection), Retrofit, Picasso, 그 외에 많은 UI 라이브러리에 대해서 공부할 수 있었고, AlarmManager, BroadcastReceiver, Service에 대해서 깊이있게 생각해볼 수 있었다.
